# Firebase Auth App (Expo + React Native)

Simple authentication app using **Firebase Auth** with **React Native** and **Expo**.  
Currently supports **Anonymous Login** — UI ready for Google and Phone logins.

---

## Features
- Anonymous Firebase authentication  
- Minimal, clean UI  
- Modular structure  
- Expo-ready project  

---

## Setup
1. Clone the repo  
   ```bash
   git clone https://github.com/yourusername/firebase-auth-app.git
   cd firebase-auth-app

2. Install dependencies

npm install


3. Add Firebase config in src/firebaseConfig.ts


4. Run

npx expo start
